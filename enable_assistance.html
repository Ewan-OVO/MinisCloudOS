# 前言

我们的远程协助服务严格遵守最小权限原则，仅用于技术人员通过安全加密通道查看设备日志，以快速定位问题根源。全程操作具备以下安全保障：

  1. 零干预承诺：技术端无法主动执行任何指令，您始终拥有设备完全控制权

  2. 无感化服务：连接期间所有用户操作不受影响，设备性能保持100%可用状态

# 1. 登录管理员账号，点击“设置”模块

# 2. 点击“管理员设置”，找到SN码

# 3. 点击“SN”，获得SN码，点击复制

# 4. 回到登录起始页，点击产品展示头图

# 5. 点击“开启远程协助”

# 6. 输入复制的SN码

# 7. 开启成功，获得远程协助码，点击复制

# 8. 将远程协助码提供给工作人员联系技术人员对接即可
